To run:

Install node.js for your operating system at:
https://nodejs.org/en/

Navigate to the location of your downloaded directory in the terminal/command prompt and type the following:

    1. 'npm install express'
    2. 'npm install socket.io'
    3. 'npm install mocha'
    4.
        4.1 'npm start' (to run the program)
        OR
        4.2 'npm test' (to run unit tests in the terminal)

    
You can then start using the program by heading to the link below in your browser:
http://localhost:8080/index.html

To terminate the program, you can use Ctrl + C in the terminal/command prompt window.
